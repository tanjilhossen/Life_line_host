স্বয়ংক্রিয় API ইন্টিগ্রেশন

আপনার হাসপাতালের স্থানীয় সফটওয়্যার বা ডেডিকেটেড অ্যাপের সাথে LifeLine যুক্ত করুন।

আপনার সিকিউর API Key

API এন্ডপয়েন্ট স্পেসিফিকেশন

METHOD: POST
ENDPOINT: /api/v1/hospital/update-beds
HEADER: x-api-key: YOUR_API_KEY

ডেভেলপার কোড এক্সাম্পল

curl -X POST http://localhost:3000/api/v1/hospital/update-beds \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "icu_available": 5,
    "emergency_bed_available": 12
  }'
fetch('http://localhost:3000/api/v1/hospital/update-beds', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    icu_available: 5,
    emergency_bed_available: 12
  })
})
.then(res => res.json())
.then(data => console.log(data));
import requests

url = "http://localhost:3000/api/v1/hospital/update-beds"
headers = {
    "x-api-key": "YOUR_API_KEY",
    "Content-Type": "application/json"
}
payload = {
    "icu_available": 5,
    "emergency_bed_available": 12
}

response = requests.post(url, headers=headers, json=payload)
print(response.json())
